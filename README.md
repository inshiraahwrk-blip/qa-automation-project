# QA Testing Portfolio

Manual and automation testing project for SauceDemo e-commerce app.

## Tech Stack
- Python 3.12
- Selenium 4
- pytest

## What's included

### Automation Testing
- Valid login — verifies redirect to inventory page
- Invalid login — verifies error handling
- URL redirect assertions
- Error handling with try/except

### Manual Testing
- 5 test cases for login page
- 3 bug reports with severity and priority
- Covers happy path, negative, and boundary scenarios

## How to run automation tests
```bash
pip install selenium
python session4_final.py
```
