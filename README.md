# QA Automation Project

Selenium automation tests for SauceDemo e-commerce app using Python.

## Tech Stack
- Python 3.12
- Selenium 4
- pytest

## What's tested
- Valid login → verifies redirect to inventory page
- Invalid login → verifies error handling
- URL redirect assertions
- Error handling with try/except

## How to run
```bash
pip install selenium
python session4_final.py
```
